# mysql-fundamentals
This repository contains structured SQL scripts covering core MySQL concepts learned through hands-on practice. 
It is organized topic-wise and focuses on building a strong foundation in relational databases for backend and full-stack development.

# Topics Covered

- Database and table creation  
- Data insertion and retrieval  
- CRUD operations (`INSERT`, `SELECT`, `UPDATE`, `DELETE`)  
- Filtering data using `WHERE`  
- Sorting and limiting results (`ORDER BY`, `LIMIT`)  
- Primary keys and foreign key constraints  
- Basic JOIN operations (`INNER JOIN`, `LEFT JOIN`)
